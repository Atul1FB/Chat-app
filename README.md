# Real-Time Chat Application 
**A full-stack real-time chat application** built with React, Node.js, Express, MongoDB, and **WebSocket (Socket.io)**.
The application allows users to **sign up**, **log in**, and **send instant messages with real-time communication**.

# Live Demo
https://chat-igz6fzzpe-chandeshwaratul148-8193s-projects.vercel.app

# Features

 User authentication (Signup / Login)
 Real-time messaging with WebSocket (Socket.io)
 Online / Offline user presence
 Unseen message notifications
 Profile image upload
 Secure password hashing
 REST API backend
 Responsive chat interface
